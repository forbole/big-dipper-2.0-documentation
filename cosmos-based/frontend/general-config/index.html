General Config Reference

The default `/src/configs/general_config.json` file should look like the following:
{  "maintainer": {    "name": "Forbole",    "url": "https://forbole.com"  },  "github": {    "reportIssue": "https://github.com/forbole/big-dipper-2.0-cosmos/issues"  }}

Type#

AttributeTypeRequiredDescriptionExample
maintainerobjecttrueDisplays who forked and is maintaining the ui-
reportIssuestringtrueDisplays where to report an explorer error on the 500 page. If you forked our report please replace it with your repo link.-
